# Bookmatrix
BookMatrix – is a modern, cloud‑ready Library Management System designed to streamline how libraries organize books, manage members, track loans, and monitor inventory. Built with a clean architecture and scalable design, BookMatrix supports multi‑client web applications, integrates seamlessly with SQL Server, and is optimized for deployment on Azure.
The platform provides a structured, efficient, and intuitive way to manage books, authors, categories, borrowing, returns, fines, and reservations — making it suitable for schools, colleges, public libraries, and private institutions.

Key Features
- Book catalog management (books, authors, categories)
- Member registration & profile management
- Borrowing, returning, and overdue tracking
- Fine calculation & payment tracking
- Reservation / hold system
- Inventory & availability management
- Role‑based access for librarians/admins
- REST API backend (ASP.NET Core)
- Angular & React client applications
- Azure‑ready deployment architecture

 Tech Stack
- Backend: ASP.NET Core Web API, EF Core, SQL Server
- Frontend: Angular & React clients
- Cloud: Azure App Service, Azure SQL, Azure Storage, Azure DevOps/GitHub Actions
